#  Tic Tac Toe Game (C++)

## Project Overview

This project is a **console-based Tic Tac Toe game** developed using C++.
It demonstrates fundamental programming concepts such as **arrays, loops, and conditional statements** through an interactive game.

##  Objectives

* To understand and implement basic C++ concepts
* To build a simple interactive game
* To practice logic building using conditions and loops


##  Features

*  2-player gameplay (Player X and Player O)
*  Dynamic board display after each move
*  Win detection (rows, columns, diagonals)
*  Draw detection
*  Replay option after game ends
*  Input validation for correct moves

##  Technologies Used

**Language:** C++

  * Arrays
  * Loops (while, for)
  * Conditional Statements (if-else)
  * Functions


##  How to Run

1. Install a C++ compiler (like g++)
2. Save the code as `main.cpp`
3. Open terminal / command prompt
4. Compile the program:
   g++ main.cpp -o tic
5. Run the program:
  ./tic

##  How to Play

* The game is played on a **3×3 grid**
* Player X goes first, followed by Player O
* Choose a position from **1 to 9**
* The first player to align 3 marks wins
* If all cells are filled → it's a draw
